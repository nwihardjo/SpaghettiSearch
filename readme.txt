COMP4321 - Search Engine
Group 12

Guideline:
1. Install golang (Refer to https://golang.org/doc/install)
2. Put the project folder to $GOPATH/src/the-SearchEngine
3. Go to the project root directory and run "go install" command on terminal
4. To start crawling, run "start_crawl"
5. To get the spider_result.txt file, run "get_result". spider_result.txt should be added on project root directory
