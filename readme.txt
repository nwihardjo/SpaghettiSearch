COMP4321 - Search Engine
Group 12

Guideline:
1. Install golang (Refer to https://golang.org/doc/install)
2. Download and put the project folder to $GOPATH/github.com/nwihardjo/the-SearchEngine
3. Go to the project root directory and run "go install"
4. To start crawling, run "start_crawl"
5. To get the spider_result.txt file, run "get_result"
